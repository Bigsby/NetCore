Installing NodeJS on Mac OS X

  1. Download packgage

    Go to NodeJS web site and click the left button, the one that says LTS (Long-Term Support).

  2. Install the package

    The package will be downloaded through the Mac OS download manager. Double-click on it.

    1. Double-click the package

    2. Introduction

      Click "Continue"

    3. "Read the license"

      Click "Continue"

    4. Confirm License

      Click "Agree"

    5. Installation Type

      Click "Install"

    6. Authenticate

      Type adminstrator password

      The actual installation takes place...

    7. And we're done!

  3. Confirm installation

    Go to the Terminal and type:

    $ node -v

    The output should be something like this:

    v4.4.5

    Also test npm is installed because it's a key component. Type:

    $ npm -v

    The output should be something like this:

    2.15.5
