Open shared (cross) project

In the previuos step, you might have noticed a folder called "consoleApp" exists in the repository. This is .Net Core Console application project, just like the one created before in these series, but created by me as myself in my physical machine.

Let's open it in Visual Studio Code.

  1. Navigate to the folder

    In the CLI:

    $ cd consoleApp

  2. Restore .Net parckages

    Still, in the CLI

    $ dotnet restore

    That should output something like this:
