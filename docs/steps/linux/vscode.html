Installing Visual Studio Code on Linux (Debian)

  1. Download packgage

    Go to Visual Studio Code web site and click ".deb" button.

    If ".deb" button doesn't show in big on the left of the page navigate to the bottom that has all platform download buttons

  2. Install package

    1. Go to the terminal and navigate to the folder where the package was downloaded to.

      cd Downloads

    2. Call Debian Package to install package

      sudo dpkg -i vscode-amd64.deb

      This should have the following output:

      Selecting previously unselected package visual-studio-code. (Reading database ... 137182 files and directories currently installed.) Preparing to unpack vscode-amd64.deb ... Unpacking visual-studio-code (1.1.1-1463146452) ... dpkg: dependency problems prevent configuration of visual-studio-code: visual-studio-code depends on git; however: Package git is not installed. dpkg: error processing package visual-studio-code (--install): dependency problems - leaving unconfigured Processing triggers for desktop-file-utils (0.22-1) ... Processing triggers for gnome-menus (3.13.3-6) ... Processing triggers for mime-support (3.58) ... Errors were encountered while processing: visual-studio-code

      So it looks like there's a dependency on Git, so...

    3. Install Git

      sudo apt-get install git

      That should also "complaint" about dependencies but, this time, with instruction to solve all of them:

      Reading package lists... Done Building dependency tree Reading state information... Try 'apt-get -f install' with no packages (or specify a solution).

      Following the instructions...

    4. sudo apt-get install -f
