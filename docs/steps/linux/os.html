Installing Linux (Debian)

For Linux I chose Debian because it's a clean distro with just what is needed for this project.

The steps:

  1. Download ISO

    Download selected version from here. "stable" amd64 network install version is used in this series.

  2. Mount ISO on machine

    In this series, a Virtual Machine is being used.

  3. Install the Operating System
