Installing .Net Core on Mac OS X

Here are the steps to install .Net Core in Mac OS X, as instructed in .Net Core web site

  1. Install Homebrew

  2. Install .Net Core package

    1. Download packgage

      Click the link that says "official PKG package"

    2. Install the package

      This will download the package through the Mac OS download manager. Double-click on it.

    3. Follow the wizard instrunctions

      Click "Continue" in "Introduction"

      Click "Continue" in "License"

      "Agree"

      Click "Install" in "Installation Type"

      Type in the password

      And were done

    Confirm installation

    Go to the Terminal and:

    $ dotnet --version

    That should output:

    1.0.0-preview1-002702
